# CSTRSIM
Continuous Stirred Tank Reactor Simulator for Fault Diagnosis Experiments

- ORIGINAL (AND CORRECTED) SIMULATOR IN FORTRAN AND PYTHON
- SIMULATOR IN C USING SIMULTANEOUS HEADS AND FLOWS
- DATA
- DOCUMENTATION
